# TaskMaster
勉強時間の見える化でいままでより効率的な学習を

## TaskMasterではこんなことができます
- タスクの名前とカテゴリーを入力することで、どんな活動をしたかがわかりやすい
- 1日にどれだけタスクに取り組んだかわかる
